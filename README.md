# Fruit Frenzy Survival

## CS50 Scratch Project

Fruit Frenzy Survival is a survival game created using Scratch for Harvard University's CS50 course.

## Objective

Survive for the entire countdown timer while avoiding the Apple and Banana enemies.

If the player survives until the timer reaches zero, the player wins.

If the player collides with an enemy, the game ends immediately.

## Controls

* W = Move Up
* A = Move Left
* S = Move Down
* D = Move Right

## Features

* WASD player movement
* Countdown timer
* Random enemy movement
* Enemy chase behavior
* Collision detection
* Game Over screen
* Victory screen
* Sound effects and background music

## How to Play

1. Start the game.
2. Use WASD keys to move the player.
3. Avoid touching the Apple and Banana.
4. Survive until the timer reaches zero.
5. Win the game by staying alive.
